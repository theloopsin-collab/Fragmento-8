# Fragmento-8

*Montado aos poucos. Intencionalmente.*

Fragmento-8 é um cyberdeck construído a partir de um Redmi Note 8.

Construído por etapas, documentado do início ao fim. Cada peça adquirida, cada configuração feita. O projeto cresce junto.

## Hardware

| Item | Modelo | Status |
|---|---|---|
| Smartphone | Redmi Note 8 | Em mãos |
| Chassi | Case rígido com espuma interna recortada | A adquirir |
| Teclado físico | Externo via Bluetooth ou USB OTG | A adquirir |
| Adaptador OTG | MicroUSB | A adquirir |

## Software

| App | Função | Onde baixar |
|---|---|---|
| Termux | Terminal Linux no Android | F-Droid |
| Termux:Styling | Temas e fontes para o terminal | F-Droid |
| Hacker's Keyboard | Teclado com teclas Ctrl/Esc/Tab | Play Store |
| Total Commander | Gerenciador de arquivos | Play Store |

Pacotes instalados no Termux:

```bash
pkg install python nmap git nano neofetch
```
